静态类 模式

实例和类

类是一些列实例的抽象，实例是类的一个具现。
实例中通常包括属性（对象的描述）和方法（对象的行为），在静态类模式下实例所拥有的属性和方法是由类模板定义的，而且定义后是不允许改变的。
```
C = Class { a, b, method }
c1 = { a: 10, b: 20 }
c2 = { a: 20, b: 30 }
```
类模式通过继承来复用代码，A继承自B，那么A就拥有B的属性和方法，B继承自C，那么A、B都有C的属性和方法，如果尝试取得A的一个属性/方法：a，那么首先在A中寻找a，如果不存在就去B中找，以此类推，如果C中也不存在则A不存在这个属性/方法。
